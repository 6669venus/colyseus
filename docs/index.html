Getting Started

First thing, make sure you have node.js v4+ installed. Colyseus uses some modern JavaScript syntax which are available on version 4+, so you might possibly need to upgrade yours.

Project structure

The simplest way to start a new project is cloning this template and install its dependencies.

To run the server, simply run npm start. The server listen on port 3553 by
default.
